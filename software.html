Software Used in XR Vehicle Simulator

The following software tools are used to create immersive, high-fidelity driving and aircraft simulations in XR environments. These tools span across development platforms, geospatial data integration, and biometric data processing.

Development Environments

Unity

Unity is a real-time development platform used to create interactive 3D experiences and simulations for XR environments.

Unreal Engine

Unreal Engine offers high-quality real-time rendering, making it ideal for building immersive and realistic simulations.

Geographic Information Systems (GIS)

ArcGIS Maps SDK for Unreal Engine

ArcGIS enables integration of geographic data, allowing us to simulate realistic terrains and environments in Unreal Engine.

Cesium for Unreal

Cesium provides high-precision 3D geospatial visualization, enabling large-scale, detailed environments within Unreal Engine.

Cesium for Unity

Cesium integrates large-scale 3D geospatial data into Unity for highly detailed environments and simulations.
