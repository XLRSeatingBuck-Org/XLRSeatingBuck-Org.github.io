XR Seating Buck

Immersive Vehicle Simulator for Ergonomic Testing and Software Validation

Explore Project

Project Highlights

Immersive Simulation

Combines Unity and Unreal Engine to deliver realistic, high-fidelity driving environments and VR experiences.

Adaptive Cockpit Hardware

Versatile seating buck designed to replicate both automotive and aircraft cockpits, with adjustable controls and modular components for ergonomic testing across multiple vehicle types.

XR Integration

Supports mixed reality headsets to simulate real-world scenarios and gather interaction data in real time.

Problem

In order to study and develop simulation technology with new hardware, we need to assemble and develop software for an extended reality seating buck. The buck itself must be modular to allow the components to easily be adjusted and swapped out. The software created for it must be developed to be easily workable with a variety of hardware. How each project attempts to accomplish this task may be found in the "Projects" tab.
